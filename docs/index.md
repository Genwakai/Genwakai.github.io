# **弦和会**

1977年に、旧)大阪女子学園高等学校マンドリン部卒業生の団体として当時顧問の宮越俊和先生が「弦和会」を発足。

現在は浅井正弘氏にご指導をいただき、マンドリン合奏音楽を愛する仲間たちが集まり大人の演奏を目標に活動をしています。

関西マンドリン連盟主催の「マンドリンフェア」に2005年より毎年参加出演しています。
